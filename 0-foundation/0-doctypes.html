This is the modern DOCTYPE: <!DOCTYPE html>

It tells browsers: “Go ahead, be modern, be relaxed. Let broken tags live.”

HTML 4.01 Strict — The Obsessive Tag Cop

Strict mode. No deprecated tags. No fun. No mercy.

HTML 4.01 Transitional — The Coward’s Choice

This let developers still use <font> and <center> while pretending to be modern.

HTML 3.2 — The Fossil Edition

This DOCTYPE is from the Netscape era. Think 1997. Painfully limited. Glorious in its horror.

XHTML 1.0 Transitional — XML Delusion

This tag pretends HTML is XML. Requires closing every tag. Slashes everywhere. Headaches inevitable.

XHTML 1.1 — The Forgotten Dictator

Almost nobody used this. Browsers didn’t even want to support it. It’s here for the masochists.
