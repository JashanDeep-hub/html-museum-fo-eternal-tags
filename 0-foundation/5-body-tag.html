This <body> tag is loaded with garbage attributes from the 1990s.

  • bgcolor: Sets background color. Ancient alternative to CSS.
  • text: Color of the normal text.
  • link: Default color of unvisited links.
  • alink: Color when link is active (being clicked).
  • vlink: Visited link color.
  • background: Sets a background image. Often tiled. Often hideous.

Modern browsers still support these... for now. Because HTML never dies, it just rots slowly.
